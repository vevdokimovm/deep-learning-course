# deep-learning-course

Jupyter notebooks with practice materials from the **Deep Learning School** course (МФТИ / ФПМИ, 2020). Covers linear regression and linear models as foundations of deep learning.

## Notebooks

| Notebook | Topic |
|----------|-------|
| `Copy_of_part_1_linear_regression.ipynb` | Linear Regression from scratch |
| `Copy_of_hw_2_linear_models.ipynb` | Homework: Linear Models |

## Tech Stack

- Python 3.x
- Jupyter Notebook
- NumPy, Pandas
- Matplotlib / Seaborn
- scikit-learn

## Getting Started

```bash
git clone https://github.com/vevdokimovm/deep-learning-course.git
cd deep-learning-course
pip install jupyter numpy pandas matplotlib scikit-learn
jupyter notebook
```
